# Hooks

В этой директории размещаются кастомные React хуки для переиспользования логики между компонентами.

## Примеры использования

- `useTasks` - хук для работы с задачами
- `useApi` - хук для работы с API
- `useForm` - хук для управления формами

## Рекомендации

- Называйте хуки с префиксом `use`
- Каждый хук должен иметь четкую ответственность
- Документируйте параметры и возвращаемые значения
